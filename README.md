# Título do Projeto

Teste DTI.

## Descrição

* Este é um projeto implementado para o teste do processo seletivo na empresa DTI utilizando a linguagem Java, a IDE Eclipse e alguns conceitos de de orientação a objeto e padrão de arquitetura de software MVC.
* Foram criadas 7 classes a fim de cumprir com os requisitos desejados e ainda garantir uma possível escalabilidade do sistema.


## Executando o Projeto

### Dependências

* Windows 10
* Java SE Runtime Environment 8

### Instalando

O projeto pode ser baixado neste link:
* [Teste DTI](https://github.com/Edd002/TesteDTI)

### Executando o Programa

* Abrir o Prompt do Window e navegar utilizando o comando [cd] até a pasta do projeto
* Executar o comando [java -jar teste-dti.jar] para executar o projeto


## Ajuda

O programa deve ser executado pelo console com o seguinte formato de entrada:

* |data| |quantidade de cães pequenos| |quantidade cães grandes|
* Exemplo: 03/08/2018 3 5
